# 증권사 API를 이용한 종목 가치 분석 프로그램
  
## 증권사 연동 기능
 - 로그인, 종목 리스트, 가격, 재무제표 정보 수집  

## 종목 별로 필요한 데이터 선정, 수집
 - PER, PBR, ROE
 - 재무제표 관련 데이터 수집(RIM, DCF)
 - 각종 마법 공식을 적용하기 위한 데이터 선정 및 수집 기능 구현

## 가치 평가 기능
 - 수집한 데이터를 기반으로 가치를 평가, sorting
 - PER, PBR, ROE등을 활용해서 동종 업종 내의 우량주와 비교해서 가치가 저평가 되어있는지 판단
